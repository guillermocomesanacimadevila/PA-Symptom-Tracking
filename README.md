# PA-Symptom-Tracking

```bash
git clone https://github.com/guillermocomesanacimadevila/PA-Symptom-Tracking.git
```

```bash
cd PA-Symptom-Tracking
```

```bash
conda env create -f env/environment.yml && conda activate symptom-tracking
```
